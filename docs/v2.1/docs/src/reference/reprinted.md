# 转载文章
 
关于转载文章在 HTML 源码中加入如下代码来指定文章作者和来源：

```html
<input id="articleAuthor" type="hidden" value="作者" />
<input id="articleSource" type="hidden" value="来源URL" />
```

::: danger 注意
是添加到 HTML 源码中，博客园文章的富文本编辑器和 Markdown 都有添加 HTML 代码的方式！
:::
